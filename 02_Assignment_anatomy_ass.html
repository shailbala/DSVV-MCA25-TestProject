Anatomy of a Web Page

1. Doctype Declaration

The doctype declaration defines the document type and must be placed at the very beginning of an HTML document.

2. HTML Element

The HTML element wraps the entire content of the web page and contains other essential elements.

3. Head Element

The head element contains meta-information about the document, such as the title, character encoding, and links to external resources like stylesheets and scripts.

4. Body Element

The body element contains the main content of the web page, including text, images, links, and other elements.

5. Header Element

The header element typically contains introductory content or navigation links for the web page.

6. Main Element

The main element represents the main content area of the web page.

7. Footer Element

The footer element contains footer information such as copyright notices, contact information, or links to related pages.
