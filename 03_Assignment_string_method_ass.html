Javascript String Method

1. String Length

The length property returns the length of a string:

#Example:
let text = "My name is Jonathan from UK"
let len = text.length;
console.log(len);
#Output: 27

2. String charAt

The charAt() method returns the character at a specified index (position) in a string:

#Example:
let text = "My name is Jonathan from UK"
let char = text.char(1);
console.log(char);
#Output: y

3. String charCodeAt

The charCodeAt() method returns the code of the character at a specified index in a string:

#Example:
let text = "My name is Jonathan from UK"
let code = text.charCodeAt(1);
console.log(code);
#Output: 121

4. String Slice (slice(start, end))

slice() extracts a part of a string and returns the extracted part in a new string:

#Example:
let text = "My name is Jonathan from UK"
let sclice = text.slice(1,6);
console.log(slice);
#Output: ?

4. String substring (substring(start, end))

substring() is similar to slice(). The difference is that start and end values less than 0 are treated as 0 in substring():

#Example:
let text = "My name is Jonathan from UK"
let sub = text.subtring(1,10);
console.log(sub);
#Output: y name is

5. String substr (substr(start, length))

substr() is similar to slice().The difference is that the second parameter specifies the length of the extracted part:

#Example:
let str = "Apple, Mango, Kiwi";
let sub = str.substr(7,5);
console.log(sub);
#Output: Mango

6. String toUppercase

A string is converted to upper case with toUpperCase():

#Example:
let text1 = "Hello World!";
let upper = text1.touppercase();
console.log(upper);
#Output: HELLO WORLD!

7. String toLowercase

A string is converted to lower case with toLowerCase():

#Example:
let text1 = "HELLO WORLD!";
let lower = text1.substr(7,6);
console.log(lower);
#Output: hello world!
