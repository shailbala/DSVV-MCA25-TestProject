HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page using a markup.

Head

The head section of an HTML document contains meta information about the document, such as its title, links to stylesheets, and scripts.

Body

The body section of an HTML document contains the content of the web page, such as text, images, videos, etc.

Header

The header element represents a group of introductory or navigational aids in a document, typically containing a group of introductory or navigational aids.

Nav

The nav element represents a section of a page that links to other pages or to parts within the page.

Main

The main element represents the main content of the document. It should be unique and not be used for repeated blocks of content, such as sidebars and navigation links.

Footer

The footer element represents a footer for its nearest sectioning content or sectioning root element. A footer typically contains information about its section, such as who wrote it, links to related documents, copyright data, etc.
