JavaScript String Methods

Method Description Example
length Returns the length of a string var str = "Hello World!";
str.length; // Returns 12
charAt() Returns the character at a specified index (position) var str = "Hello";
str.charAt(0); // Returns 'H'
charCodeAt() Returns the Unicode of the character at a specified index var str = "Hello";
str.charCodeAt(0); // Returns 72
[ ] Returns the character at a specified index (position) var str = "Hello";
str[0]; // Returns 'H'
slice() Extracts a section of a string and returns it as a new string var str = "Hello World!";
str.slice(0, 5); // Returns "Hello"
substring() Returns the characters between two specified indices of a string var str = "Hello World!";
str.substring(0, 5); // Returns "Hello"
substr() Returns the characters in a string beginning at the specified location through the specified number of characters var str = "Hello World!";
str.substr(6, 5); // Returns "World"
toUpperCase() Converts a string to uppercase letters var str = "Hello";
str.toUpperCase(); // Returns "HELLO"
toLowerCase() Converts a string to lowercase letters var str = "Hello";
str.toLowerCase(); // Returns "hello"
concat() Joins two or more strings var str1 = "Hello";
var str2 = "World!";
str1.concat(" ", str2); // Returns "Hello World!"
trim() Removes whitespace from both ends of a string var str = " Hello ";
str.trim(); // Returns "Hello"
trimStart() Removes whitespace from the beginning of a string var str = " Hello";
str.trimStart(); // Returns "Hello"
trimEnd() Removes whitespace from the end of a string var str = "Hello ";
str.trimEnd(); // Returns "Hello"
padStart() Pads the current string with another string until the resulting string reaches the given length var str = "Hello";
str.padStart(10, "X"); // Returns "XXXXXHello"
padEnd() Pads the current string with another string until the resulting string reaches the given length var str = "Hello";
str.padEnd(10, "X"); // Returns "HelloXXXXX"
repeat() Returns a new string with a specified number of copies of the string it was called on var str = "Hello";
str.repeat(3); // Returns "HelloHelloHello"
replace() Returns a new string with some or all matches of a pattern replaced by a replacement var str = "Hello World!";
str.replace("World", "Universe"); // Returns "Hello Universe!"
replaceAll() Returns a new string with all matches of a pattern replaced by a replacement var str = "Hello World!";
str.replaceAll("o", "x"); // Returns "Hellx Wxrld!"
split() Splits a string into an array of substrings var str = "Hello World!";
str.split(" "); // Returns ["Hello", "World!"]
