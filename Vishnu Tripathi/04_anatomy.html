Anatomy of a Webpage

<!DOCTYPE html>

The <!DOCTYPE html> declaration represents the document type and helps the browser render the page correctly.

<html></html>

The <html> element is the root element that encompasses the entire HTML document.

<head></head>

The <head> element contains metadata about the document, such as the title, stylesheets, scripts, and other information not visible on the webpage itself.

<head> <title>Anatomy of a Webpage</title> <link rel="stylesheet" href="styles.css"> <script src="script.js"></script> </head>

<body></body>

The <body> element contains the visible content of the webpage, such as headings, paragraphs, images, and other HTML elements.

<body> <h1>Welcome to my website!</h1> <p>This is a sample paragraph.</p> <img src="image.jpg" alt="Sample Image"> </body>

HTML Elements

HTML elements are the building blocks of a webpage. They are represented by tags enclosed in angle brackets, like <p> for a paragraph or <h1> for a top-level heading.

CSS

Cascading Style Sheets (CSS) are used to control the presentation and styling of HTML elements. CSS rules are typically defined in a separate file and linked to the HTML document via the <link> element in the <head> section.

JavaScript

JavaScript is a programming language used to add interactivity and dynamic behavior to webpages. JavaScript code is either embedded directly in the HTML document or included as an external file using the <script> element, typically in the <head> or <body> section.
