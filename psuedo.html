CSS Pseudo-classes and Pseudo-elements

Pseudo-class/Element Description Example
::after Insert content after an element
::before Insert content before an element
::first-letter Selects the first letter of an element
::first-line

Selects the first line of an element

Selects the first line of an element
::selection Selects the portion of an element that is selected by a user
:active Selects the active link
:checked Selects any radio button or checkbox that is checked
:disabled Selects any disabled element
:empty Selects any element that has no children
:first-child Selects the first child element of its parent
:focus Selects the element that has focus
:hover Selects an element when you mouse over it
:lang(language) Selects every element in a document that is written in a specific language
:last-child Selects the last child element of its parent
:link Selects all unvisited links
:not(selector) Selects every element that is not a specified element
:nth-child(n) Selects every element that is the nth child, regardless of type, of its parent
:nth-last-child(n) Selects every element that is the nth child, regardless of type, of its parent, counting from the last child
:nth-of-type(n) Selects every element that is the nth child of its parent, of a specific type
:nth-last-of-type(n) Selects every element that is the nth child of its parent, of a specific type, counting from the last child
:only-child Selects every element that is the only child of its parent
:only-of-type Selects every element that is the only child of its parent, of a specific type
:root Selects the document's root element
:target Selects the current active #target element
:visited Selects all visited links
